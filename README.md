# overtrain-site

Source for the [overtrain](https://github.com/) GitHub Pages
microsite - landing page, privacy policy, and FAQ for the
`overtrain` Android indoor-cycling app.

Static Markdown rendered by GitHub Pages with the built-in
`jekyll-theme-minimal` theme - no local build step.

## Editing
- 
- `index.md` — landing page
- `privacy.md` — privacy policy
- `faq.md` — FAQ + troubleshooting
- `_config.yml` — Jekyll theme + site title
